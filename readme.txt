INSTRUCCIONES PUESTA EN MARCHA PROYECTO
    1. Descargue el repositorio
    2. Para XAMPP, guarde el repositorio en la raiz de la aplicacion (C:\xampp\htdocs\...) creando una carpeta con el nombre del proyecto.*
    3. Abra el XAMPP y ponga a correr los servicios de APACHE y MySQL.
    4. En su navegador de preferencia, abra el cliente de MySQL (http://localhost/phpmyadmin/) y ejecute el query que aparece en el archivo 'sql_query.sql', para crear la DataBase y las Tablas necesarias para correr el sitio.
    5. En su navegador, ingrese al sitio http://localhost/xxxxx/index.php donde 'xxxxx' es el nombre de la carpeta de proyecto creada en el paso 2 de estas instrucciones.
